# PKI Lab

This repository documents my hands-on journey of designing and implementing a Public Key Infrastructure (PKI) using OpenSSL on Ubuntu Server 24.04.

The objective is to build a solid understanding of PKI concepts and apply them in a practical enterprise context, progressing from a Root Certificate Authority (CA) to Intermediate CAs, certificate issuance, revocation and validation.

## Topics Covered

- PKI fundamentals
- Root Certificate Authority (CA)
- Intermediate Certificate Authority (CA)
- Certificate Signing Requests (CSRs)
- Server and client certificates
- Certificate chains
- Certificate Revocation Lists (CRLs)
- Online Certificate Status Protocol (OCSP)
- Enterprise PKI best practices

## Lab Environment

- Ubuntu Server 24.04 LTS
- OpenSSL
- PNETLab
- Cisco Security technologies (ISE, ASA, etc.)
