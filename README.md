# CCIE Security Lab

This repository documents my hands-on journey towards developing enterprise infrastructure, networking and security engineering skills for CCIE Security level.

The focus is on building practical experience through real-world labs, documenting concepts, implementation procedures and lessons learned along the way.

## Repository Structure

| Folder | Description |
|--------|-------------|
| **0-PNETLab-Audit** | Scripts and tools for auditing and maintaining the PNETLab environment. |
| **1-PKI-Lab** | Building an enterprise Public Key Infrastructure (PKI) using OpenSSL on Ubuntu, from Root CA to certificate issuance and validation. |

Additional labs will be added as the repository grows, covering Cisco security technologies, cloud networking, automation and enterprise infrastructure.

## Objectives

- Build practical enterprise networking and security skills.
- Develop repeatable implementation and troubleshooting procedures.
- Document concepts clearly and concisely.
- Produce reusable scripts, configuration templates and technical documentation.
- Create a professional engineering portfolio demonstrating practical experience.
